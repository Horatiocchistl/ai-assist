# Gap Types

Use these gap_type values when writing findings. Choose the one that best describes what you observed.

**image_missing** — A planned asset does not appear anywhere on the live page. The position is empty or filled with something unrelated.

**image_wrong** — An image appears in the expected position but it is clearly a different product, scene, or subject than what was planned.

**color_mismatch** — The same or similar image appears but the colors look noticeably different — background, product color, overlay color.

**layout_shift** — The composition or arrangement has changed. Elements are in different positions, cropped differently, or the proportions have shifted.

**asset_swapped** — The same general asset appears but it looks like a different version — updated artwork, different variant, different crop of the same shoot.

**extra_image** — An image appears in the carousel or A+ that was not in the planned assets. This may or may not be a problem — note it and describe what it shows.

**copy_drift** — Live text (title, bullets, description) differs from the planned copy specification. Note exactly what differs.

**text_on_image_diff** — Text that appears within an image (overlay text, callout, badge) differs between the planned and live versions.

**order_mismatch** — Images appear in a different sequence than planned. Note the planned order vs the live order.

**content_missing** — An A+ module, bullet point, or section that was planned is entirely absent from the live page.

**custom_finding** — Used for findings from CPG custom skill modules that do not fit the above categories.
