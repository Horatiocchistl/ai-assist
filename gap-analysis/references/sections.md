# Page Sections

These are the sections of an Amazon product detail page you will analyze:

**hero** — The main product image shown at the top of the page. This is the first image a shopper sees. It must clearly show the product against the correct background with the correct branding.

**carousel** — The sequence of additional product images shown in the image carousel. Order matters — the planned sequence reflects the brand's intended story. Note each image's position and content.

**aplus** — A+ Content modules below the fold. These are brand-designed sections with images and text telling the product story. Layout, imagery, and copy all matter here.

**copy** — The text content of the listing: title, bullet points, description, and specs. This is compared against the approved copy specification.

When you call `list_gap_sections`, the tool tells you which sections are actually present in the live captures for this ASIN. Only analyze sections that exist.
